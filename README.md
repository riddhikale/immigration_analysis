Project Overview
This project analyzes Canadian immigration data by country of citizenship from 1980 to 2013 using Python, Pandas, NumPy, and Jupyter Notebook.
The notebook focuses on data cleaning, exploration, filtering, aggregation, and basic analysis to uncover immigration trends across countries, continents, and regions.

Insights & Analysis

- Identified top 5 countries contributing to Canadian immigration
- Sorted countries based on total immigration
- Observed regional and continental migration patterns

Key Findings

- India and China are the largest contributors to immigration in Canada over the period.
- Asian countries dominate overall immigration numbers.
- Immigration trends vary significantly across regions and time periods.
